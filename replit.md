# Alanallur Borewell & Pumps Website

## Overview
A fully responsive multi-page business website for Alanallur Borewell & Pumps, a Kerala Government approved borewell contractor. The website showcases their services, projects, and provides contact options for customers.

## Project Structure
```
/
├── index.html          # Home page with hero, highlights, services, testimonials
├── about.html          # Company info, team, values, certifications
├── services.html       # Detailed service descriptions (7 services)
├── projects.html       # Completed project showcase with timeline
├── gallery.html        # Photo gallery with filters and lightbox
├── faq.html            # Collapsible FAQ accordion
├── contact.html        # Contact form, map, and contact details
├── styles.css          # Global styles with CSS variables
├── script.js           # JavaScript for navigation, lightbox, form validation
└── replit.md           # This documentation file
```

## Technologies Used
- Pure HTML5, CSS3, and Vanilla JavaScript (no frameworks)
- Responsive design with CSS Grid and Flexbox
- CSS variables for theming
- SVG icons for scalability

## Features
- Fully responsive for mobile, tablet, and desktop
- Mobile hamburger navigation menu
- Floating WhatsApp button for quick contact
- Gallery with category filters and lightbox
- FAQ with collapsible accordion
- Contact form with client-side validation
- Scroll animations (fade-in effects)
- SEO meta tags and Open Graph tags
